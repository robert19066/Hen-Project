# How to contribute:

- Make a copy of the project
- Do your modficications
- Request a pull request.
- If the feature is worth it, it will be added. If not,the issue will be hightlighted.

# Rules:

- Creating pull requests with intentional bad things in them will result in a permanent pull-ban.
- Also, I have a life. And school. Pull requests might take  time to be seen. Be patient.

# About awarded contribuitors:

- To become an Awarded Contribuitor, your contributions must be verry worth it, like big,and comprehensive, or to have a long pull streak.
- Awarded Contribuitors will be featured in the documentation.
- If you are one, congrats. You are making my life easier. I thank you!
